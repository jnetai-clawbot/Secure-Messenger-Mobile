v1.0.1 released: https://github.com/jnetai-clawbot/Secure-Messenger-Mobile/releases
Changes:

- No central server — true P2P with built-in TCP/UDP networking
- Default Google STUN: stun.l.google.com:19302, stun1.l.google.com:19302

- STUN-based NAT traversal for public address discovery
- Same keystore — updates without uninstall





Secure Mobile Messenger v1.0.0 is built and released.
- Release: https://github.com/jnetai-clawbot/Secure-Messenger-Mobile/releases
- APK: Secure Mobile Messenger.apk (~3MB)
- Repo: https://github.com/jnetai-clawbot/Secure-Messenger-Mobile
What's included:
- P2P pairing via QR code & 6-digit PIN with regenerate button
- Per-contact RSA-2048 + AES-256-GCM encryption (unique keys per contact)
- 1-to-1 encrypted chat with file sharing, nudge, block, end chat, clear history
- Settings: network, chat, storage location, themes (dark/light + custom colors)
- About section with "Made by jnetai.com", version, update check, share
- Room database, DataStore preferences, persistent DebugLogger with error codes
- GitHub Actions CI/CD with same keystore for updates
